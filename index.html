CHROMASYNC is a local-network app that reads your currently playing Spotify track, extracts the dominant album art color, and syncs it to your Govee smart bulbs in real time — with smooth exponential color transitions between songs.

How it works
01
Spotify reads now playing
Polls the Spotify Web API every second to detect track changes, progress, and queue — without hammering the API.
02
Color extracted from art
Album art is quantized using Pillow to find the most vibrant dominant color, routed through RGBWW logic for the H6008's 5-channel LED array.
03
Bulbs sync over LAN
Color commands are sent via the Govee LAN UDP API with per-device rate limiting. Natural track endings trigger smooth exponential blends.
